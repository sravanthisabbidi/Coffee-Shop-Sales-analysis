# ☕ Coffee Shop Sales Analysis (Excel Project)

## 📌 Project Overview
This project analyzes coffee shop sales data using Microsoft Excel to uncover key business insights and trends.

## 🛠 Tools Used
- Microsoft Excel
- Pivot Tables
- Charts & Dashboard

## 📊 Key Insights
- Identified top-selling products
- Analyzed sales by time (daily/monthly trends)
- Revenue contribution by category
- Peak sales hours

## 📈 Dashboard
![Dashboard](dashboard/dashboard_screenshot.png)

## 📁 Dataset
The dataset includes:
- Order ID
- Product Name
- Category
- Quantity
- Price
- Date & Time

## 💡 Business Recommendations
- Focus on high-performing products
- Optimize staffing during peak hours
- Promote low-performing items
